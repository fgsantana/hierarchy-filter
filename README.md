## How to run
### [Inside the project folder](https://github.com/fgsantana/hierarchy-filter/blob/main/hierarchy-filter):
- Testing all cases:
> ./gradlew test

- Selecting the test case to run:
> ./gradlew test --tests 'FilterTests.{test case title}'
